# Advanced Events (Pty) Ltd — Smart Event Management & Ticketing Platform

## Overview
A full-stack, production-grade event management and ticketing web application built with Node.js, Express, EJS, MongoDB, and Mongoose.

## Tech Stack
- **Runtime**: Node.js
- **Framework**: Express.js
- **Templating**: EJS with express-ejs-layouts
- **Database**: MongoDB + Mongoose
- **Auth**: express-session + bcrypt
- **Uploads**: Multer
- **QR Codes**: qrcode
- **PDF**: pdfkit
- **Email**: nodemailer
- **Charts**: Chart.js (frontend)
- **Styling**: Custom CSS (no frameworks, fully bespoke)

## Quick Start
```bash
npm install
cp .env.example .env
# Fill in your .env values
npm run dev
```
